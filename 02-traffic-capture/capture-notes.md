# Network Traffic Capture

## Interface Used
- eth0 / wlan0 (depending on VM configuration)

## Applied Filters
- HTTP:
  http
- DNS:
  dns

## Procedure
1. Wireshark was launched on the Kali Linux machine.
2. The active network interface was selected.
3. Web traffic was generated by accessing the vulnerable application.
4. DNS queries were performed from the target system.
5. Captures were saved in `.pcapng` format.

## Included Files
- capture-http-login.pcapng
- capture-dns-traffic.pcapng
